Timelog — Layout-Optionen

Welches Layout passt besser zur Arbeitsweise? Inspiriert von Lecterns Design-Sprache.

A

Single-Page Dashboard

Ein View: Timer prominent oben, aktive Activity, Stille-Indikator, Einträge darunter. Keine Navigation. Alles auf einen Blick. Minimal.

B

Sidebar + Content (wie Lectern)

Linke Sidebar mit Navigation: Timer, Heute, Woche, Projekte. Hauptbereich zeigt den jeweiligen View. Skalierbarer für mehr Features.

C

Timer-Fokus + Slide-Over

Timer dominiert den Screen (groß, zentriert). Reports/Einstellungen als Slide-Over Panel von rechts. Timer bleibt immer sichtbar.
